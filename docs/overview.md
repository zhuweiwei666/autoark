# AutoArk Project Overview

AutoArk is an automated advertising optimization platform. It connects to advertising networks like Facebook and TikTok to fetch data, analyze performance, and execute automated optimization rules.

## Key Components

- **Backend**: Node.js + Express + TypeScript
- **Database**: MongoDB Atlas
- **Frontend**: React (Planned)
- **AI Engine**: LLM-based optimization suggestions
- **Rule Engine**: Automated actions based on performance metrics
